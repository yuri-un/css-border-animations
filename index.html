Using Border Colors

Pros: easy to implement, CSS style attributes manipulation.

Cons: limited animation posibilities.

Using Border Styles

Pros: easy to implement, CSS style attributes manipulation.

Cons: limited animation posibilities.

Using Border Sides

Pros: easy to implement, CSS style attributes manipulation.

Cons: limited animation posibilities.

Using Border Image

Pros: allows animations with different geometrical primitives and other uncommon animation styles, mediocre complexity, relatively difficult to implement.

Cons: requires at least 4 image frames for a smooth animation, border styles can't be changed with CSS styles (even color).

Using Background Image

Pros: allows complex animation effects, mediocre complexity, easy to implement.

Cons: requires decent @keyframes frames for a smooth animation.

Using SVG

Pros: allows complex animation effects, mediocre complexity, easy to implement.

Cons: requires decent @keyframes frames for a smooth animation.

Using Nested Divs

Pros: allows complex animation effects, mediocre complexity, easy to implement.

Cons: requires decent @keyframes frames for a smooth animation.

Using Outline Style

Pros: allows complex animation effects, mediocre complexity, easy to implement.

Cons: requires decent @keyframes frames for a smooth animation.

Using Box Shadow

Pros: allows complex animation effects, mediocre complexity, easy to implement.

Cons: requires decent @keyframes frames for a smooth animation.
